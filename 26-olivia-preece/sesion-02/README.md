### GITHUB ###  
Plataforma en la nube diseñada para desarrolladores, que permite almacenar, gestionar y compartir código fuente utilizando el sistema de control de versiones Git. Funciona como una "red social" para programadores, facilitando la colaboración, el seguimiento de cambios y el desarrollo de proyectos de softwar, tanto públicos (código abierto) como privados.

### MARKDOWN ###  
Es un lenguaje de marcado ligero. En términos simples, es una forma de escribir texto con formato (negrita, títulos, listas, incluir enlaces o imágenes) añadiendo unas simples marcas al texto, usando solo caracteres comunes del teclado, sin necesidad de botones complicados o menús ocultos.

### Saltos de líneas ###  
*   Párrafo nuevo: Enter dos veces.
*   Bajar una línea: Dos espacios al final + Enter
*   Forzar el salto: <br>

### Encabezados ###  
* # Título Nivel 1
* ## Título 2
* ### Título 3
* así continuamente.

### Formato de texto ###  
* Negrita -> **Texto** o __texto__
* Cursiva -> *texto* o _texto_
* Negrita y cursiva -> ***Texto***
* Tachado ->
* Escapado de símbolos -> \#texto

### Listas ###

* NÚMEROS
